###苹果电脑清理桌面缓存

http://www.macgg.com/archives/24126.html

1. cd  /private/var/vm

2. sudo pmset -a hibernatemode 0 

3. sudo pmset -a autopoweroff 0

4. sudo pmset -a standby 0

5. sudo rm /var/vm/sleepimage 

6. purge  相当于关机重启
